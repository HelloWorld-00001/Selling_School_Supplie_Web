1. Middle ware: là folder lưu những file trung gian, giữ các file controller dùng để tương tác với db hoặc views
2. Models: là folder chứa các file tương tác với db
3. Public: là folder chứa các file css và hình ảnh liên quan
4. routes: là folder chứa các file sử dụng các model và hiển thị lên UI 
5. Ultis: là folder sử dụng và kết nối với db
6. views: là folder chứa các file thiết kế và hiển thị UI, tương tác với user -- team front end chủ yếu sẽ làm ở đây
7. file app: là file xử lí chính: chỉ gọi các hàm ra thôi :))
8. file json: là mấy file cấu hình thôi :))
